Install Extensions

Click extension icon on the left panel of VS Code, then search for the extension and click install.
Further details can be set and selected on the extension page and settings.

Necessary

  • Live Server

    • Enables live browsing of the html file that updates automatically after each saving action
    To use, right click on the html file from explorer and select 'Open With Live Server'.
    Usual practice is to split the screen between the VS Code editor and the browser.

  • Prettier

    • Auto formats the html file upon saving.
    Needs to be configured and enabled in the text editor format settings.

  • AutoRename Tag

    • Helpful when changing tag items, specially when renaming an opening tag, the closing
    tag automatically changes too.

Optional

  • Material Theme

    • Be able to select colors theme for the VS code.

  • Material Icon

    • Use some common and standard icons.

Boiler Plate

When creating a new and empty html file, the boiler plate can be used.
Besides easing up the typing task, the standard html format can be observed.

  • How To
    1. Create a new html file from the explorer.
    2. In the empty file beginning at line 1, type Shift+1 or '!'.
    3. Then press 'TAB' key, and the html boiler plate is ready.

Keyboard Shortcuts

  • Line Copy and Paste

    • A line of code can be duplicated by pressing SHIFT+ALT and ArrowUP key to copy
    above or with ArrowDown to copy below the current line.

  • Comment/Uncomment

    • Select a line or group of lines, then press CTRL and '/' to comment or uncomment it.

  • Insert A new Line Below

    • Inserting a new line by pressing ENTER key depends on your current cursor location.
    If the cursor is somewhere inside a line of text, pressing ENTER breaks that line.
    To avoid this, press CTRL and ENTER
